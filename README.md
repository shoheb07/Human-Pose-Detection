# Human-Pose-Detection
The Human Pose Detection System is a computer vision project developed using Python, OpenCV, and MediaPipe. The application detects and tracks human body movements in real-time through a webcam.

# Human Pose Detection

## Project Overview

The Human Pose Detection project detects and tracks human body landmarks in real-time using a webcam.

The project uses MediaPipe Pose and OpenCV for AI-based body tracking.


# Features

- Real-time pose detection
- Body skeleton tracking
- Webcam integration
- AI-based landmark detection


# Technologies Used

- Python
- OpenCV
- MediaPipe

# Installation

Install required libraries:

pip install opencv-python mediapipe


# Run Project

python pose_detection.py


# How It Works

1. Capture webcam video
2. Detect body landmarks
3. Draw pose skeleton
4. Display live tracking output
